CSS Básico - Temario Completo

CSS separa presentación de estructura HTML, permitiendo consistencia visual, mantenibilidad y reutilización.

selector { propiedad: valor; }

En producción se prioriza CSS externo para modularidad y caché eficiente.

<link rel="stylesheet" href="styles.css">
* { box-sizing: border-box; } #menu { ... } .card > h3 { ... } input[type="email"] { ... }

Usa rem para tipografía base y %/vw para diseños fluidos.

html { font-size: 16px; } h1 { font-size: 2rem; } .container { width: min(90%, 1100px); }
:root { --brand: #0a4d68; --accent: rgb(5 191 219); } .hero { background: linear-gradient(135deg, var(--brand), #088395); }
p { text-align: justify; line-height: 1.6; letter-spacing: 0.02em; text-transform: none; }
body { font-family: 'Inter', sans-serif; } h1 { font-family: 'Orbitron', sans-serif; font-weight: 900; }
ul.custom { list-style: square inside; padding-left: 1rem; } li::marker { color: #088395; }
table { border-collapse: collapse; width: 100%; } th, td { border: 1px solid #dbe4ee; padding: .75rem; } th { background: #0a4d68; color: white; }
* { box-sizing: border-box; } .card { margin: 1rem; padding: 1rem; border: 1px solid #cfd8e3; }
.panel { width: clamp(280px, 60vw, 900px); min-height: 200px; max-height: 60vh; }
.fixed-help { position: fixed; right: 1rem; bottom: 1rem; } .sticky-nav { position: sticky; top: 0; } .badge { position: absolute; top: .5rem; right: .5rem; }
a:hover { color: #05bfdb; } input:focus { outline: 2px solid #088395; } .card::before { content: ''; display: block; height: 4px; background: #05bfdb; }
.code-block { overflow-x: auto; overflow-y: hidden; } .modal-body { max-height: 70vh; overflow: auto; }
input, select, textarea { width: 100%; padding: .7rem; border: 1px solid #cfd8e3; border-radius: .5rem; } input:invalid { border-color: #ff6b35; }
  • Define variables en :root para colores y espaciado.
  • Evita selectores demasiado específicos.
  • Organiza por capas: base, layout, componentes, utilidades.
  • Prioriza accesibilidad y contraste.

BEM: bloque, elemento y modificador para escalabilidad en equipos.

.card {} .card__title {} .card--featured {}

Curso de Programación CSS3 + Emmet

Ruta rápida para practicar sintaxis CSS3 con abreviaturas Emmet y generar estructura HTML/CSS más rápido.

1) Sintaxis CSS3 esencial

/* Regla básica */ selector { propiedad: valor; } /* Agrupación y cascada */ h1, h2, h3 { color: #0a4d68; } /* Variables */ :root { --brand: #05bfdb; } .btn { background: var(--brand); }

2) Selectores útiles

/* Clase, ID, atributo, descendiente */ .card {} #menu {} input[type="email"] {} .layout .item {} /* Estados */ a:hover {} button:focus-visible {} li:nth-child(2n) {}

3) Emmet para maquetar rápido

Escribe la abreviatura en VS Code y presiona Tab.

nav.navbar>ul.nav-list>li.nav-item*5>a{Sección $} section.hero>h1{Curso CSS3}+p{Aprende sintaxis y práctica} main.container>section.card*3>h2{Tema $}+p{Contenido} form.form>input:text+input:email+button.btn{Enviar} div.grid>article.card*6>h3{Card $}+p{Texto}

4) Emmet para CSS

m10 -> margin: 10px; p20 -> padding: 20px; bgc#0a4d68 -> background-color: #0a4d68; df -> display: flex; jcc -> justify-content: center; aic -> align-items: center; g20 -> gap: 20px; bd1#ccc -> border: 1px solid #ccc; bdr12 -> border-radius: 12px; fz1.2rem -> font-size: 1.2rem;

5) Mini práctica guiada

/* Objetivo: tarjeta responsive + hover */ .card-demo { width: min(100%, 320px); padding: 1rem; border: 1px solid #dbe4ee; border-radius: .8rem; transition: transform .25s ease, box-shadow .25s ease; } .card-demo:hover { transform: translateY(-6px); box-shadow: 0 10px 24px rgba(0,0,0,.16); }

10 Ejercicios Prácticos Guiados

Bloque de práctica progresiva para afianzar la Unidad 7. Cada ejercicio incluye objetivo técnico y una solución base para comparar.

Estiliza todos los párrafos dentro de .articulo y solo el primer párrafo de cada artículo.

.articulo p { color: #0a4d68; } .articulo p:first-of-type { font-weight: 700; border-left: 4px solid #05bfdb; padding-left: .6rem; }

Define un estilo base para botones y crea un modificador para botón crítico sin usar !important.

.btn { background: #0a4d68; color: #fff; } .btn--danger { background: #d90429; }

Crea tipografía escalable usando rem y bloque fluido con clamp().

html { font-size: 16px; } h1 { font-size: clamp(1.8rem, 2vw + 1rem, 3rem); } .contenedor { width: min(92%, 1100px); }

Define paleta en :root y aplica gradiente al hero.

:root { --bg-1: #0a4d68; --bg-2: #088395; } .hero { background: linear-gradient(135deg, var(--bg-1), var(--bg-2)); color: #fff; }

Construye una tarjeta con padding interno, borde y sombra consistente.

* { box-sizing: border-box; } .card { padding: 1rem; margin: 1rem 0; border: 1px solid #dbe4ee; border-radius: 12px; box-shadow: 0 6px 14px rgba(0,0,0,.08); }

Alinea título a la izquierda y botones a la derecha en una misma fila.

.toolbar { display: flex; justify-content: space-between; align-items: center; gap: .75rem; }

Construye una rejilla que se adapte automáticamente al ancho disponible.

.grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(240px, 1fr)); gap: 1rem; }

Aplica estados hover/focus-visible para navegación con teclado.

a:hover { color: #05bfdb; } button:focus-visible, a:focus-visible { outline: 3px solid #05bfdb; outline-offset: 2px; }

Estiliza inputs válidos e inválidos sin JavaScript.

input { border: 1px solid #cfd8e3; border-radius: .5rem; padding: .65rem; } input:valid { border-color: #00c896; } input:invalid { border-color: #ff6b35; }

Refactoriza una card usando bloque, elemento y modificador.

.card {} .card__title {} .card__meta {} .card--destacada { border-color: #05bfdb; }

Práctica Final Integrada

Construye una landing completa aplicando todos los temas: tipografía, tablas, formularios, flex/grid, pseudoclases, overflow y nomenclatura BEM.

Checklist: +- Variables globales CSS +- Layout responsive (Grid + Flex) +- Formulario con estados :focus y :invalid +- Tabla con estilo semántico +- Clases BEM en componentes +- Buen contraste y accesibilidad

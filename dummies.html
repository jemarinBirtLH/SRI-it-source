Ruta de aprendizaje

Cada modulo sube dificultad poco a poco. Si un nivel te rompe, respira, toma agua y vuelve al anterior.

Modulo 1: Setup (1-5)

Instalacion de VS Code, carpetas, orden de proyecto y primeros archivos.

Modulo 2: HTML (6-20)

Estructura, semantica, tablas, formularios y mini landing sin CSS.

Modulo 3: CSS (21-35)

Color, tipografia, flex, grid, responsive y estilo limpio sin sufrimiento.

Modulo 4: JavaScript (36-50)

DOM, eventos, validacion, localStorage, fetch y proyecto final interactivo.

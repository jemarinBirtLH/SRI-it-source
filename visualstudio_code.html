Visual Studio Code

Editor multiplataforma, gratuito y extensible para cualquier lenguaje.

Descargar y recursos oficiales

  • Web oficial de Visual Studio Code (versiones para Windows, macOS y Linux).
  • Documentación oficial.
  • Repositorio en GitHub (código abierto).

Extensiones más utilizadas

  • Prettier - formateo de código automático.
  • ESLint - linting de JavaScript/TypeScript.
  • Live Server - servidor web local con recarga en vivo.
  • Python - soporte oficial para Python.
  • GitLens - supercarga de git integrada.
  • Bracket Pair Colorizer / Bracket Pair Colorizer 2 - colorea pares de llaves.
  • Colorize - muestra colores en CSS/SCSS y más.
  • Remote - SSH / WSL / Containers - desarrollo remoto.

Temas de color y personalización

Instala temas populares como One Dark Pro, Dracula Official, Solarized o Night Owl. Cambia el icono y la configuración de la barra lateral desde Settings > Color Theme.

Asistentes de código y AI

  • Copilot (requiere suscripción de GitHub) para autocompletar basado en IA.
  • Codex a través de extensiones no oficiales que consumen la API de OpenAI.
  • Ollama y otros asistentes locales con modelos de lenguaje.
  • Tabnine - completado AI multi-modelo.

Referencias y ejemplos

Puedes encontrar snippets de configuración en Snippets y aprender atajos por sistema en Key Bindings. Ejemplos de settings.json:

{
    "editor.formatOnSave": true,
    "files.exclude": { "**/.git": true },
    "workbench.colorTheme": "One Dark Pro",
    "editor.tabSize": 2
}

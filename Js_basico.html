JavaScript Basico

Fundamentos universitarios: sintaxis, DOM, asincronia, modulos y practicas reales.

1) Sintaxis y tipos

JavaScript trabaja con tipado dinamico. Usa const por defecto y let cuando el valor cambie.

const alumno = "Jemarin"; let nota = 9.2; const aprobado = nota >= 5; const temas = ["variables", "funciones", "dom"]; const perfil = { nombre: alumno, nota, aprobado };

2) Estructuras de control y funciones

function clasificarNota(nota) { if (nota >= 9) return "sobresaliente"; if (nota >= 7) return "notable"; if (nota >= 5) return "aprobado"; return "suspenso"; } const resumen = (nombre, nota) => `${nombre}: ${clasificarNota(nota)}`;

4) Asincronia y Event Loop

Microtareas (Promises) se ejecutan antes de macrotareas (setTimeout).

console.log('A'); setTimeout(() => console.log('B'), 0); Promise.resolve().then(() => console.log('C')); console.log('D'); // salida: A, D, C, B

JavaScript a Nivel Universitario

Modelo de ejecución, asincronía, modularidad y patrones de arquitectura.

1. Modelo de Ejecución y Event Loop

JavaScript ejecuta código en un hilo principal con cola de tareas. Entender microtareas (Promises) y macrotareas (timers) es clave para diseñar interfaces reactivas sin bloqueos.

console.log('A'); setTimeout(() => console.log('B'), 0); Promise.resolve().then(() => console.log('C')); console.log('D'); // Salida: A, D, C, B

2. Diseño Modular y Escalable

Separar dominio, infraestructura y presentación reduce acoplamiento. Usa módulos ES y funciones puras para lógica crítica.

// domain/ip.js export function isValidIpv4(ip) { const re = /^(25[0-5]|2[0-4]\d|1\d\d|[1-9]?\d)(\.(25[0-5]|2[0-4]\d|1\d\d|[1-9]?\d)){3}$/; return re.test(ip); }

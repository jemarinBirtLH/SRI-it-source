Ollama + Chatbox AI (local y privado)

Ollama

Empresa: OllamaTipo: Local AI Runtime

Motor para ejecutar modelos en tu propio equipo (CPU/GPU), sin depender siempre de la nube.

Descarga: ollama.com/download

# Instalar modelo y probar ollama pull llama3.1 ollama run llama3.1 # API local compatible HTTP curl http://localhost:11434/api/generate -d "{\"model\":\"llama3.1\",\"prompt\":\"Hola\"}"
Web oficial

Chatbox AI

Empresa: Chatbox TeamTipo: Cliente Desktop/Web

Interfaz tipo chat para conectar modelos de OpenAI, Claude, Gemini, DeepSeek y Ollama local.

Descarga: chatboxai.app

Integración con Ollama: 1) Inicia Ollama (puerto local 11434) 2) En Chatbox: Settings - Model Provider - Ollama 3) Base URL: http://127.0.0.1:11434 4) Elige modelo (ej. llama3.1, qwen3, deepseek-r1)
Abrir Chatbox

Tutorial rápido de instalación OpenClaw

Instalación orientada a CLI (según documentación pública):

# 1) Instalar (macOS/Linux/WSL2) curl -fsSL https://openclaw.ai/install.sh | bash # En Windows PowerShell (alternativa) iwr -useb https://openclaw.ai/install.ps1 | iex # 2) Asistente de configuración openclaw onboard --install-daemon # 3) Comprobar estado del gateway openclaw gateway status # 4) Abrir panel local openclaw dashboard

Tip: en Windows normalmente se recomienda WSL2 para una instalación más estable.
